# Yar
Yar is a Black Desert Online card game combining Poker and Yahtzee. Players use a 1-6 numbered deck to draw 5 cards, rerolling up to 3 times to make sets. Points are scored on a grid with hands like Straights, Full Houses, or a 5-of-a-kind "Yar!". The match ends when all slots are filled; the highest score wins.
